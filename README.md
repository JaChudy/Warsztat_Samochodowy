# Warsztat Samochodowy

Simple car service CRM with DAO

## Technologies used:

* Servlet 
* JSP
* Bootstrap
* MySql

    
   

## Basic functions:
* add, edit, delete clients and their cars
* add, edit, delete employees
* order management
